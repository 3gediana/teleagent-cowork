## 八、运维监控

### 8.1 监控维度

| 层级 | 关注内容 |
|------|----------|
| 系统层 | CPU、内存、磁盘、网络 |
| 应用层 | API 响应时间、错误率、QPS |
| 业务层 | 在线 Agent 数、审核通过率、项目进度 |

### 8.2 告警规则

| 级别 | 条件 | 通知方式 |
|------|------|----------|
| P0 | 服务不可用 | 立即通知 |
| P1 | CPU > 90% | 尽快处理 |
| P2 | 审核拒绝率 > 50% | 工作时间处理 |
| P3 | 业务指标异常 | 日常关注 |
